# 🏥 AI-Based Medical Test Triage & Specialist Recommendation System

An intelligent system that assists in early medical diagnosis by analyzing patient data and recommending appropriate medical tests and specialist consultations using Machine Learning.

---

## 📌 Project Overview

This project aims to build a machine learning-based system that:

- Analyzes patient medical data  
- Predicts the likelihood of diseases  
- Assists in triaging patients for appropriate medical tests  
- Recommends relevant medical specialists  

The system is designed to work on multiple medical datasets to improve robustness and generalization.

---

## 📊 Datasets Used

This project utilizes **multiple medical datasets** to enhance prediction capability:

- **Chronic Kidney Disease (CKD)**
- **Heart Disease Dataset**
- **Diabetes Dataset**
- **Liver Disease Dataset**

These datasets include both **numerical** and **categorical** features such as:

- Blood Pressure  
- Glucose Levels  
- Hemoglobin  
- Age  
- Medical conditions (Hypertension, Diabetes, etc.)
